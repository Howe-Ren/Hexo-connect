NewBing-signup

NewBing注册

  • 准备工作：
    1. 魔法工具（VPN）。 2. Microsoft Edge Dev浏览器。 3. 微软账号。 4. 申请NewBing资格。
  • 主要操作：
    1. 下载浏览器
      • 下载链接：Download Microsoft Edge Insider Channels
      • 点击“Download”下载即可
      • 之后界面选择浏览器的语言，可以“English”或列表底部的“简体中文”
    2. 登录微软账号
      • 打开安装好的浏览器，点击左上角头像进入登录界面
      • 填写个人账号信息
    3. 申请NewBing
      • 成功登录后，进入申请网页链接：The New Bing - Learn More
      • 点击“加入等待列表”或“Join the waitlist”（我这边已经加入，所以中间是“ChatNow”）
    4. 打开NewBing新世界
      • 点击蓝色“b”气泡图标，展开侧边栏，输入即可开始对话！
      • 也可在浏览器搜索框搜索，然后 鼠标滚轮上翻/触摸屏上翻 进入BingChat

ChatGPT SignUp and Usage

  • 注册过程
    • 邮箱取号
      • 选择一个可用的电子邮箱，去注册OpenAi账号（国内的qq邮箱测试可行）
    • 国外电话号码验证
      • 选择一个可以接受短信的国外电话号码的平台，并注册（推进较慢）；购买一个地区的号码资格（我选择的平台上，一个号码的有效期是20min；购买后行动未成功，可在20min内取消该号码，平台会全额退款）
      • 购买前需要先充值1美元（支持支付宝，加手续费约7.03人民币）
      • 第一个号码选的马来西亚，后来其他方面的操作有延迟，所以退了；之后选择了印度尼西亚的号码（较便宜，30卢布左右），成功！
  • 开始对话
    • 注册成功后（邮箱+电话验证），在Chat网站登录账号
    • 有可能会出现“信息过载”之类的报错警告，这个报错可能是因为自己使用的VPN节点不干净，可以更换纯净节点（用的人少的），继续尝试登录
